# Public

Repo-tracked public narrative and go-to-market working artifacts for Goalrail.

This layer exists before any automation or publishing workflow.
Humans update it manually until a real operating surface exists.

## Layout

- `public/stories/` — durable narrative arcs
- `public/posts/` — concrete draft/final post assets
- `public/channels/` — channel notes and constraints
- `public/publications/` — publication receipts
- `public/metrics/` — manual performance snapshots
- `public/_templates/` — starter templates
- `public/_schema/` — future metadata/schema helpers

Public material must not outrun the canonical product docs.
