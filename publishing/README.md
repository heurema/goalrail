# Publishing

Repo-tracked public narrative and go-to-market working artifacts for Goalrail.

This is the publication plane, not a frontend static-assets `public/` directory.

This layer exists before any automation or publishing workflow.
Humans update it manually until a real operating surface exists.

## Layout

- `publishing/stories/` — durable narrative arcs
- `publishing/posts/` — concrete draft/final post assets
- `publishing/channels/` — channel notes and constraints
- `publishing/publications/` — publication receipts
- `publishing/metrics/` — manual performance snapshots
- `publishing/_templates/` — starter templates
- `publishing/_schema/` — future metadata/schema helpers

Public material must not outrun the canonical product docs.
