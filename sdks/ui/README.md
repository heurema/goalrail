# goalrail-ui-sdk

Terminal UI components (Rich + prompt_toolkit) for building
[Goalrail](https://github.com/heurema/goalrail) frontends.

`goalrail-ui-sdk` provides the reusable rendering and input building blocks used
by the Goalrail CLI to display streaming agent output in the terminal. It is
released in lockstep with the core `goalrail` package at a matching version:

```bash
pip install goalrail-ui-sdk
```

See the [Goalrail repository](https://github.com/heurema/goalrail) for full
documentation.
