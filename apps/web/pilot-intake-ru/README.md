# Goalrail Pilot Intake RU

Russian pilot-intake landing prototype for Goalrail.

## Scope

- Static React + Vite + Mantine resource under `apps/web/pilot-intake-ru`
- Intended for the Russian landing domain `goalrail.ru`; the English landing should be a separate resource later
- Implements the `Goalrail Pilot Intake.html` design handoff
- Keeps the page as a lead-capture/public landing prototype, not a product runtime

## Commands

Run from `apps/web`:

```bash
npm run pilot-intake-ru:dev
npm run pilot-intake-ru:test
npm run pilot-intake-ru:typecheck
npm run pilot-intake-ru:build
```
