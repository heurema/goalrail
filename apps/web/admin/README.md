# Goalrail Admin Web

Real Goalrail admin shell.

Target subdomain: `console.goalrail.dev`.

Current scope:
- left navigation with three empty product surfaces: Contracts, Delivery Readiness, Proof
- no data, API calls, backend integration, auth, routing, persistence, or CLI/server status
- visual shell only, based on the existing change-packet demo design language

Delivery rule:
- CLI and server functionality should become real first
- admin cards and detail views should appear only after the underlying functionality exists
