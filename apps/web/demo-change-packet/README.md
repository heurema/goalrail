# Demo Change Packet Web

React implementation of the Anthropic design handoff for `Goalrail Change Packet Demo.html`.

## Source handoff

Primary source file used for the implementation:

- `Goalrail Change Packet Demo.html` from the handoff bundle

## Stack

- React
- Vite
- Mantine runtime shell
- Plain CSS for pixel-oriented demo styling
- Vitest + React Testing Library

## Commands

From `apps/web`:

```bash
npm run demo-change-packet:dev
npm run demo-change-packet:build
npm run demo-change-packet:test
npm run demo-change-packet:typecheck
```

## Notes

This is a prototype demo surface for the change packet walkthrough.
It is intentionally a demo shell, not a claim that the full Goalrail product loop is implemented.
