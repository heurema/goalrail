# GOALRAIL — Reference Decision

## Reference #1

Use **Factory.ai** as the main external reference for:

- product seriousness
- model-agnostic supplement-layer framing
- enterprise-ready tone
- "platform above existing tools" narrative

## What to borrow

- calm, serious positioning
- provider-agnostic framing
- supplement layer over existing tools
- mature product tone

## What NOT to copy

- agent-centric core promise
- "delegate complete tasks to droids" framing
- making the agent the hero instead of the shared contract

## Goalrail distinction

Goalrail should remain centered on:

- business intent
- shared contract
- bounded execution
- verification + proof
